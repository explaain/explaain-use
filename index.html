Use explaain

Add explaain cards to your website

To use explaain cards on your website, just add this line of code.

<script src="https://use.explaain.com/explaain.js"></script>

This script is cross browser, has no dependancies and is under the namespace "explaain" so it won't clash with JavaScript on your site.

How to use explaain

1) Display related cards by keywords

This example displays cards matching the keywords "Donald Trump".

HTML

<div class="explaain" data-keywords="Donald Trump"></div>

Example

2) Display a specific card by ID

This example displays a specific card using it's ID.

HTML

<div class="explaain" data-id="https://explaain-api-develop.herokuapp.com/Organization/5787d1f6dac9821100609b1f"></div>
