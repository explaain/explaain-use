Use Explaain

Add Explaain cards to your website

To use Explaain cards on your website, just add this line of code.

<script src="http://use.explaain.com/explaain.js"></script>

This script is cross browser, has no dependancies and is under the namespace "explaain" so it won't clash with JavaScript on your site.

Alternatively, to view Explaain temporarily, one page at a time, in your own browser only, use the Explaain bookmarklet:

Use Explaain

Simply drag this button into your bookmarks bar, and click it at any time to Explaain-ify the webpage you're viewing!

How to use Explaain

1) How to link to cards in-line

You can also also link to cards in text and it will detect the links.

HTML

<p>
  Excepteur sint occaecat cupidatat non proident, sunt in culpa
  <a href="http://app.explaain.com/Person/57b31ba4c38e87110021a439">Theresa May</a>
  qui officia deserunt mollit anim id est laborum.
</p>

Example

Excepteur sint occaecat cupidatat non proident, sunt in culpa Barry White qui officia deserunt mollit anim id est laborum.

Excepteur sint occaecat cupidatat non proident, sunt in culpa Kevin Bacon qui officia deserunt mollit anim id est laborum.

Excepteur sint occaecat cupidatat non proident, sunt in culpa Theresa May qui officia deserunt mollit anim id est laborum.

Excepteur sint occaecat cupidatat non proident, sunt in culpa Boris Johnson qui officia deserunt mollit anim id est laborum.

2) Display related cards by keywords (coming soon!)

This example displays cards matching the keywords "Theresa May".

HTML

<div class="explaain" data-keywords="Theresa May"></div>
